## 8. متطلبات التكامل الخارجي (Integration)

### 8.1 التكاملات الحالية

| الخدمة | الغرض | طريقة الاتصال |
|--------|-------|---------------|
| Supabase | قاعدة البيانات + التخزين + المصادقة | @supabase/supabase-js SDK |
| Google Gemini 2.0 Flash | المحادثة الذكية + البحث + التوصيات + توليد الأوصاف | @google/genai SDK |

### 8.2 التكاملات الجديدة

| الخدمة | الغرض | طريقة الاتصال |
|--------|-------|---------------|
| **Moamalat** | بوابة الدفع الإلكتروني (بطاقات، تحويل، mVisa) | LightBox JavaScript SDK |
| **Vanex** | خدمة التوصيل والشحن في ليبيا | REST API (تفاصيل Endpoints ستُحدد لاحقاً) |

### 8.3 تفاصيل تكامل Moamalat

```
البيئة التجريبية: https://tnpg.moamalat.net:6006/js/lightbox.js
البيئة الإنتاجية: https://npg.moamalat.net:6006/js/lightbox.js

المعاملات المطلوبة:
- MID: معرف التاجر (11-18 حرف)
- TID: معرف المحطة (6-8 حرف)
- AmountTrxn: المبلغ بأصغر وحدة (1 دينار = 1000)
- SecureHash: HMAC SHA-256 (64 حرف)
- TrxDateTime: التاريخ بتنسيق yyyyMMddHHmm
- رمز العملة: 434 (الدينار الليبي)

طرق الدفع المدعومة:
- البطاقات البنكية (Card)
- التحويل (Tahweel)
- mVisa
```

### 8.4 تفاصيل تكامل Vanex (placeholder)

```
الموقع الرسمي: vanex.ly
التوثيق: docs.vanex.ly (Swagger UI)
Postman Collection: متاح على Postman API Network

Endpoints المتوقعة (سيتم تحديدها):
- إنشاء شحنة (Create Shipment)
- تتبع الشحنة (Track Shipment)
- قائمة المدن/المناطق (Get Cities/Zones)
- حساب تكلفة الشحن (Calculate Price)
- إلغاء الشحنة (Cancel Shipment)
- Webhooks لتحديثات الحالة (Status Updates)
```
